# CIFAR-10 Image Classification

Решение задачи "Классификация изображений CIFAR-10" с помощью нейронной сети на языке Python с использованием библиотеки TensorFlow.

## Запуск кода

```bash
uv sync
uv run main.py
```

## Аргументы запуска

### Параметры данных

`--data-dir`: Папка с данными CIFAR-10 (по умолчанию "data") \

### Параметры модели

`--batch-size`: Размер батча (по умолчанию 32) \
`--embedding-dim`: Размерность эмбеддинга (по умолчанию 256) \
`--learning-rate`: Learning rate оптимизатора (по умолчанию 0.001)

### Параметры обучения

`--epochs`: Количество эпох обучения (по умолчанию 20) \
`--validation-split`: Доля валидации (по умолчанию 0.1) \
`--verbose`: Уровень логов (по умолчанию 1)

### Параметры сохранения

`--model-dir`: Папка для сохранения модели (по умолчанию "results") \
`--results-dir`: Папка для сохранения результатов (по умолчанию "results") \
`--model-name`: Имя файла модели (по умолчанию "cifar10_model")
